​​ចំណាំ​​អំពី​​ការ​​ដំឡើង
----------------------------------------------------------------------

LibreOffice ទាមទារ​កំណែ​ថ្មី​មួយ​របស់ Java Runtime Environment (JRE) ដើម្បី​ឲ្យ​មាន​មុខងារ​ពេញលេញ ។ JRE មិន​មែន​ជា​ផ្នែក​នៃ​កញ្ចប់​ដំឡើង​របស់ LibreOffice ទេ វា​គួរតែ​ត្រូវ​បាន​ដំឡើង​ដាច់​ដោយ​ឡែក ។

តម្រូវ​ការ​ប្រព័ន្ធ​
----------------------------------------------------------------------

* Microsoft Windows 7 SP1, 8, 8.1 Update (S14) or 10

សូម​ស្វែងយល់​ថា សិទ្ធិ​អ្នក​គ្រប់គ្រង​ត្រូវ​បាន​ទាមទារ​សម្រាប់​ដំណើរការ​ដំឡើង ។

ការ​ចុះ​ឈ្មោះ LibreOffice ជា​កម្មវិធី​លំនាំ​ដើម​សម្រាប់​ទ្រង់ទ្រាយ Microsoft Office អាច​ត្រូវ​បាន​បង្ខំ ឬ ទប់ស្កាត់​ដោយ​ប្រើ​បន្ទាត់​ពាក្យ​បញ្ជា​ខាងក្រោម​ប្ដូរ​ជាមួយ​កម្មវិធី​ដំឡើង ៖

* REGISTER_ALL_MSO_TYPES=1 នឹង​បង្ខំ​ចុះ​ឈ្មោះ LibreOffice ជា​កម្មវិធី​លំនាំ​ដើម​សម្រាប់​ទ្រង់ទ្រាយ Microsoft Office ។
* REGISTER_NO_MSO_TYPES=1 នឹង​បង្ហាប់​ការ​ចុះ​ឈ្មោះ LibreOffice ជា​កម្មវិធី​លំនាំ​ដើម​សម្រាប់​ទ្រង់ទ្រាយ Microsoft Office ។

សូម​ប្រាកដ​ថា​ អ្នក​មាន​អង្គ​ចងចាំ​ទំនេរ​គ្រប់គ្រាន់​​ ក្នុង​ថត​​បណ្ដោះ​អាសន្ន​លើ​ប្រព័ន្ធ​របស់​អ្នក ​ហើយ​សូម​ប្រាកដ​ថា​ការ​​អាន​ ការ​សរសេរ​ និង​សិទ្ធ​ចូល​ដំណើរការ​ត្រូវ​បាន​អនុញ្ញាត​ ។​ បិទ​គ្រប់​កម្មវិធី​ផ្សេង​សិន​​ មុន​ពេល​ចាប់ផ្ដើម​ដំណើរការ​ដំឡើង​ ។

ការ​ដំឡើង​ LibreOffice លើ​ប្រព័ន្ធ​លីនុច​ដែល​មាន​មូលដ្ឋាន​លើ​​​ ដេបៀន​/អ៊ូប៊ុនទូ​
----------------------------------------------------------------------

សម្រាប់​សេចក្ដី​ណែនាំ​អំពី​របៀប​ដំឡើង​កញ្ចប់​ភាសា (បន្ទាប់ពី​ដំឡើង LibreOffice កំណែ​ភាសា​អង់គ្លេស US) រួច, សូម​អាន​ផ្នែក​ដែល​នៅ​ខាងក្រោម​កញ្ចប់​ភាសា​ការ​ដំឡើង​ដែល​បាន​ផ្ដល់​សិទ្ធិ។

ពេល​អ្នក​ស្រាយ​ប័ណ្ណសារ​ដែល​បាន​ទាញ​យក​ អ្នក​នឹង​ឃើញ​មាតិកា​ដែល​មិន​បាន​បង្ហាប់​​​ទៅ​ក្នុង​ថត​រង។ បើក​បង្អួច​កម្មវិធី​គ្រប់គ្រង​ឯកសារ​​​ រួច​ប្ដូរ​ទៅកាន់​ថត​មួយ​ដែល​ចាប់ផ្ដើម​ដោយ​ពាក្យ "LibreOffice_" និង​តាម​តាម​ពី​ក្រោយ​ដោយ​​លេខ​កំណែ​ និង​ព័ត៌មាន​​កម្មវិធី​មួយ​ចំនួន។

ថត​នេះ​មាន​ថត​រង​មួយ​ ​ហៅ​ថា​ "DEBS" ។ ប្ដូរ​ថត​ទៅ​ជា​ថត​ "DEBS" ។

ចុច​​ខាង​​ស្ដាំ ខាង​ក្នុង​ថត​ ហើយ​ជ្រើស​​ "បើក​​ក្នុង​ស្ថានីយ" ។​ បង្អួច​ស្ថានីយ​នឹង​បើក​ ។​ ពី​បន្ទាត់​ពាក្យ​បញ្ជា​ស្ថានីយ​​របស់​បង្អួច​ស្ថានីយ​​ បញ្ចូល​ពាក្យ​បញ្ជា​ខាង​ក្រោម​​ (អ្នក​នឹង​បាន​ដាក់​ប្រអប់​​បញ្ចូល ដើម្បី​បញ្ចូល​ពាក្យ​សម្ងាត់​អ្នក​ប្រើ​របស់​​អ្នក​ជា​​ root ​មុន​​​ពាក្យ​បញ្ជា​​នឹង​ប្រតិបត្តិ) ។

ពាក្យ​បញ្ជា​ខាងក្រោម​នឹង​ដំឡើង​ LibreOffice និង​កញ្ចប់​ផ្ទៃតុ (អ្នក​គ្រាន់តែ​ចម្លង និង​បិទភ្ជាប់​វា​ទៅ​ក្នុង​អេក្រង់​ស្ថានីយ​ជាជាង​ការ​វាយ​ពួក​វា)៖

sudo dpkg -i *.deb

ឥឡូវ​ បាន​បញ្ចប់​ដំណើរការ​ដំឡើង​ អ្នក​គួរ​តែ​មាន​រូបតំណាង​​ សម្រាប់​​គ្រប់​កម្មវិធី​ LibreOffice ក្នុង​ម៉ឺនុយ​ កម្មវិធី​​/Office ផ្ទៃ​តុ​របស់​អ្នក​ ។

ការ​ដំឡើង LibreOffice នៅ​លើ Fedora, openSUSE, Mandriva និង​នៅ​លើ​ប្រព័ន្ធ​លីនុច​ផ្សេងទៀត​ដោយ​ប្រើ​កញ្ចប់ RPM

សម្រាប់​សេចក្ដី​ណែនាំ​អំពី​​របៀប​ដំឡើង​កញ្ចប់​ភាសា (បន្ទាប់ពី​ដំឡើង LibreOffice កំណែ​ភាសា​អង់គ្លេស US), សូម​អាន​ផ្នែក​ដែល​នៅ​ក្រោម​កញ្ចប់​ភាសា​ដំឡើង​ដែល​បាន​ផ្ដល់​សិទ្ធិ។

នៅ​ពេល​អ្នក​ស្រាយ​ប័ណ្ណសារ​ដែល​បាន​ទាញ​យក​ អ្នក​នឹង​ឃើញ​មាតិកា​ដែល​មិន​បាន​បង្ហាប់​​​ទៅ​ក្នុង​ថត​រង ។ បើក​បង្អួច​កម្មវិធី​គ្រប់គ្រង​ឯកសារ រួច​ប្ដូរ​ថត​ទៅកាន់​ថត​មួយ​ដែល​ចាប់ផ្ដើម​ជាមួយ "LibreOffice_" ហើយ​តាម​ពី​ក្រោយ​ដោយ​លេខ​កំណែ​ និង​ព័ត៌មាន​​កម្មវិធី​មួយ​ចំនួន​ ។

ថត​នេះ​មាន​ថត​រង​ដែល​ហៅថា​ "RPMS" ។​ ប្ដូរ​ថត​ទៅ​ជា​ថត​ "RPMS" វិញ​ ។

ចុច​​ខាង​​ស្ដាំ ខាង​ក្នុង​ថត​ ហើយ​ជ្រើស​​ "បើក​​ក្នុង​ស្ថានីយ" ។​ បង្អួច​ស្ថានីយ​នឹង​បើក​ ។​ ពី​បន្ទាត់​ពាក្យ​បញ្ជា​ស្ថានីយ​​របស់​បង្អួច​ស្ថានីយ​​ បញ្ចូល​ពាក្យ​បញ្ជា​ខាង​ក្រោម​​ (អ្នក​នឹង​បាន​ដាក់​ប្រអប់​​បញ្ចូល ដើម្បី​បញ្ចូល​ពាក្យ​សម្ងាត់​អ្នក​ប្រើ​របស់​​អ្នក​ជា​​ root ​មុន​​​ពាក្យ​បញ្ជា​​នឹង​ប្រតិបត្តិ) ។

Problems When Sending Documents as Emails From LibreOffice
----------------------------------------------------------------------

When sending a document via 'File - Send - Document as Email' or 'Document as PDF Attachment' problems might occur (program crashes or hangs). This is due to the Windows system file "Mapi" (Messaging Application Programming Interface) which causes problems in some file versions. Unfortunately, the problem cannot be narrowed down to a certain version number. For more information visit https://www.microsoft.com to search the Microsoft Knowledge Base for "mapi dll".
